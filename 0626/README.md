## 1. 입력과 형변환

`input()`은 사용자가 입력한 값을 문자열로 받습니다.

그래서 숫자 계산을 하려면 `int()` 또는 `float()`로 바꿔야 합니다.

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))
```

이후 f-string을 사용해서 계산 결과를 보기 좋게 출력했습니다.

```python
print(f"a + b = {a + b}")
print(f"a - b = {a - b}")
print(f"a * b = {a * b}")
print(f"a / b = {a / b}")
```

### 핵심 정리

| 코드 | 의미 |
|---|---|
| `input()` | 사용자 입력 받기. 결과는 문자열 |
| `int()` | 문자열을 정수로 변환 |
| `float()` | 문자열을 실수로 변환 |
| `str()` | 숫자 등을 문자열로 변환 |
| `f"{변수}"` | 변수나 계산 결과를 문자열 안에 넣어 출력 |

## 2. 리스트에 값 저장하기

`1.py`에서는 정수 5개를 입력받아 리스트에 저장했습니다.

```python
y = []

for i in range(5):
    a = int(input("5개의 정수값을 입력하시오: "))
    y.append(a)
```

여기서 중요한 흐름은 다음과 같습니다.

```text
빈 리스트 생성 → 반복문으로 입력 받기 → append로 저장 → 리스트 전체 계산
```

---

## 3. 리스트 기본 통계

리스트에 숫자가 들어 있으면 Python 내장 함수로 간단한 통계를 구할 수 있습니다.

```python
print(f"mean = {sum(y) / len(y)}, sum = {sum(y)}, min = {min(y)}, max = {max(y)}")
```

| 함수 | 의미 |
|---|---|
| `len(y)` | 리스트 길이 |
| `sum(y)` | 리스트 값들의 합계 |
| `min(y)` | 최솟값 |
| `max(y)` | 최댓값 |

평균은 직접 `sum(y) / len(y)`처럼 계산했습니다.

## 4. 문자열 리스트와 인덱싱

`2.py`에서는 과일 이름을 입력받아 리스트에 저장했습니다.

```python
fruits = []

for i in range(5):
    fruit = input("enter a fruit: ")
    fruits.append(fruit)
```

리스트는 순서가 있기 때문에 인덱스로 값을 꺼낼 수 있습니다.

```python
print(fruits[-1])
print(fruits[3])
```

### 인덱싱 정리

| 코드 | 의미 |
|---|---|
| `fruits[0]` | 첫 번째 값 |
| `fruits[3]` | 네 번째 값 |
| `fruits[-1]` | 마지막 값 |

Python 인덱스는 0부터 시작한다는 점이 중요합니다.

## 오늘 배운 핵심

- `input()` 결과는 문자열이다.
- 숫자 계산을 하려면 `int()` 또는 `float()`가 필요하다.
- f-string을 사용하면 변수와 계산식을 쉽게 출력할 수 있다.
- 리스트는 여러 값을 순서대로 저장하는 자료구조이다.
- `append()`는 리스트 끝에 값을 추가한다.
- `sum`, `len`, `min`, `max`로 리스트의 기본 계산을 할 수 있다.
- 리스트 인덱스는 0부터 시작한다.
- 음수 인덱스 `-1`은 리스트의 마지막 값을 의미한다.
